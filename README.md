# unity-docs-translate

translate the English unity docs to Other languages

## how to use

1. init the project

```bash
bun install
```

2. download the doc

```bash
bun run download
```

3. translate the doc

```bash
bun run translate
```

4. you can use for MCP,RAG,etc.
